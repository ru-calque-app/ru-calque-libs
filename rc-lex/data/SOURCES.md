# Источники словаря `lexicon.tsv`

Файл собран `tools/build_lexicon.py` из четырёх открытых источников. Все они допускают
коммерческое использование при указании авторства — атрибуция ниже обязательна и должна
ехать вместе с продуктом.

| Источник | Что даёт | Лицензия | Обязательство |
|---|---|---|---|
| [CEFR-J Vocabulary Profile 1.5](https://github.com/openlanguageprofiles/olp-en-cefrj) (Tono Lab, TUFS) | 7 798 слов + POS + уровень A1–B2 | Свободно для исследовательского и **коммерческого** использования при цитировании | Указать CEFR-J / Tono Lab |
| [Octanove Vocabulary Profile C1/C2 1.0](https://github.com/openlanguageprofiles/olp-en-cefrj) | 2 136 слов уровня C1–C2 | CC BY-SA 4.0 | Указать автора; share-alike на производную БД при её распространении |
| [FrequencyWords](https://github.com/hermitdave/FrequencyWords) (OpenSubtitles 2018, en_50k) | частотность → `zipf` | CC BY-SA 4.0 | Указать автора |
| [Wiktionary](https://en.wiktionary.org) (category API: English phrasal verbs, English idioms) | 5 147 фразовых глаголов, 9 851 идиома | CC BY-SA 4.0 + GFDL | Указать Wiktionary |

## Что сознательно НЕ используется

Проверено и отклонено — лицензии не позволяют:

- **EFLLex** (CENTAL/UCLouvain) — CC BY-**NC**-SA, только некоммерческое;
- **KELLY** — CC BY-NC-SA, то же;
- **English Vocabulary Profile** (Cambridge) — проприетарный, несмотря на то что он лучший
  из существующих (уровень на каждое значение, включая фразовые глаголы). Любые его копии
  на Kaggle/GitHub — тоже нельзя: это скраны;
- **Oxford 3000/5000** — © OUP, все права защищены.

Если однажды понадобится настоящий уровень для фразовых глаголов и коллокаций — лицензию
на EVP надо покупать у Cambridge, обходных путей нет.

## Пересборка

```bash
python3 tools/build_lexicon.py              # качает источники, пишет data/lexicon.tsv
python3 tools/build_lexicon.py --cache /tmp/lex   # переиспользовать скачанное
```

Результат коммитится: словарь вкомпилирован в крейт (`include_str!`), сеть в рантайме не
нужна. Wiktionary отдаёт ~30 страниц по 500 записей — сборка занимает минуту.

## Чему в этом файле нельзя доверять как разметке

Колонка `cefr_source`:

- `cefrj` — уровень взят из размеченного людьми словаря, это факт;
- `derived` — уровень **выведен нами** (`derive_mwe_cefr`: уровень головного слова + одна
  ступень, либо по частотности). У многословных единиц уровня нет ни в одном открытом
  источнике, поэтому иначе никак — но это оценка, и строить на ней жёсткие решения нельзя.

Колонка `zipf` у многословных единиц всегда `0.00`: частотный список униграммный, частоты
биграмм в нём нет.
